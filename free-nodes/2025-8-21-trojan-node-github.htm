✅ 特点：提供治疗、增益和护盾技能，增强队友战斗能力。
✅ 战斗定位：确保队伍持久作战，增强团队续航能力。
✅ 适用策略：灵活走位，远离战场中心，专注支援队友。

4. 坦克（Tank）——铜墙铁壁，吸收伤害

✅ 特点：血量高，承伤能力强，适合吸引敌方火力。
